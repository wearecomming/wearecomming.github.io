Analysis of Sqlalchemy Table Structure

I divided the database into four interrelated data tables, namely User, Record, ImageFile, and Achievement

The following is a table structure description and details

Table Name describe content
User User table, used to store user related information username, password, phone number
Record Consumption and expenditure record table, used to track user’s various income and expenditure details billing name, date, amount, merchant, remarks
ImageFile Image table, used to save all images, including ticket images, etc. Image name, image data
Achievement Achievement table, used to record the achievements made by users in accounting Achievement name, Achievement description
